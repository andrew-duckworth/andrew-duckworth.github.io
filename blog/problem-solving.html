This is when I used google as a problem solving method to search how to loop through an array within an array and found some examples on stack overflow and advice on the Dev Academy discord channel that helped me solve the problem. At first I felt a bit frustrated but kept looking for solutions. I learned that being able to use other peoples experiences online can help my own learning.
